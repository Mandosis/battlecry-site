# Star Wars: Battlecry Website

## Setup
### Requirements
* Node.js (v6.2.1)
* RethinkDB (v2.3)

### Instructions
1. `git clone https://github.com/Mandosis/battlecry-site.git`

2. `cd battlecry-site`

3. `nano config.json` or `vim config.json` and change settings as needed.

4. `npm install`

5. `npm start`
